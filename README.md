# 아지트 안내글 템플릿 시스템
## 1. 개발 배경 및 목적
당사 서비스데스크에서는 사내 커뮤니케이션 툴인 아지트(Agit)를 이용하여 자신 지급/반납, 퇴직/휴직, IT기기나 소프트웨어 트러블슈팅 등 여러 업무를 처리한다. 문의자들이 아지트에 문의글을 올리면 담당자들이 전산 처리 후 댓글로 안내글을 적는다. 이 과정에서 댓글에는 담당자 멘션, 반납 기한 안내, 자산별 부속품 여부, 네트워크 등록/삭제 여부 등 포함해야 할 정보가 많은데, 이러한 정보는 또 문의에 따라 제각각이기 때문에 그 수는 일일히 사람이 기억하고 작성하기에는 한두가지 정도 빠트릴 정도로 많다. 때문에 안내글을 적을 때 아지트에 같은 문의글을 찾아 그 글에 달린 댓글을 복사한 뒤 내용을 수정해서 작성하는 방식을 사용해 왔는데, 해당 방법의 경우에는 날짜 오기입이나 담당자 누락이 발생하기 쉽고, 비슷한 문의글을 찾는 작업에 대한 리소스 낭비도 크다. 이를 해결하고자 브라우저에서 즉시 접근 가능한 사이드 패널 형태의 크롬 확장 프로그램을 개발하여 몇번의 클릭과 입력만으로 표준화된 안내글 양식을 자동 생성해주는 시스템을 개발하게 되었다.

## 2. 프로젝트 개요
* 사내 자산관리시스템과 연계하여 즉시 아지트 안내 메세지를 생성하고 복사할 수 있는 크롬 확장 프로그램
* 브라우저 사이드 패널로 동작하여 다른 화면을 가리지 않고 실시간으로 입력값을 동기화하여 안내글 서식을 완성해주는 생산성 향상 툴
* 직관적인 UI와 원클릭 자산 정보 스크래핑 기능
  
## 3. 개발환경
* 프론트엔드 : HTML, JavaScript, Tailwind CSS
* 확장 프로그램 API : Chrome Manifest V3

## 4. 사용 기술
### 프론트엔드
* HTML/JS (바닐라)
  * `document.addEventListener` 및 실시간 DOM 조작(`syncLogic`)을 통해 폼 입력값을 하단 프리뷰 영역에 즉각 렌더링
  * `ClipboardItem` API를 활용하여 생성된 템플릿을 아지트에 서식(링크, 인용구 등) 깨짐 없이 `text/html` 형태로 복사
* Tailwind CSS
  * CDN 방식으로 적용하여 별도의 빌드 과정 없이 빠른 UI 스타일링
  * 사이드 패널 환경에 최적화된 레이아웃(너비 600px 기준 3단 뷰) 구성

### 확장 프로그램 API
* `sidePanel` API: 확장 프로그램 아이콘 클릭 시 브라우저 우측에 고정 패널을 오픈하여 작업 편의성 제공 (`openPanelOnActionClick`)
* `scripting` & `activeTab API`: 현재 열려 있는 탭(자산 관리 시스템 페이지)의 DOM에 접근하여 선택된 자산의 품명 및 코드를 스크래핑하는 웹 훅 역할 수행

## 6. 설치 방법
1. 본 저장소의 코드를 로컬 PC로 다운로드(ZIP 다운로드 또는 git clone)하고 압축을 해제
2. 크롬 브라우저 주소창에 chrome://extensions/를 입력하여 확장 프로그램 관리 페이지로 이동
3. 화면 우측 상단에 있는 '개발자 모드' 토글을 클릭하여 활성화
4. 화면 좌측 상단에 나타난 '압축해제된 확장 앱을 로드합니다.' 버튼을 클릭
5. 앞서 다운로드하여 압축을 해제한 소스코드 폴더 자체를 선택
6. 브라우저 우측 상단의 퍼즐 조각 아이콘(🧩)을 클릭한 뒤, '아지트 템플릿 마스터' 옆의 고정(핀셋 📌) 아이콘을 눌러 주소창 옆에 항상 보이도록 설정
